This section contains steering files that provide context to Claude Code and team members.

## What are Steering Files?

Steering files define project conventions, coding standards, and contextual information that help AI assistants (like Claude Code) and new team members understand how the project works.

## How to Use

- Add subpages under this section for different topics
- Claude Code will reference these when working on the project
- Keep them up to date as conventions evolve

## Suggested Topics

- **Coding Standards** — naming conventions, formatting rules, patterns to use/avoid
- **Architecture** — system design, data flow, key decisions and trade-offs
- **Team** — who works on what, responsibilities, communication channels
- **Deployment** — environments, CI/CD, release process
- **Company Context** — product goals, customer needs, business constraints

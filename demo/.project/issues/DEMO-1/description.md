# Set up CI/CD pipeline

Configure a complete CI/CD pipeline for the project using GitHub Actions.

## Requirements

- Run tests on every PR
- Build and deploy to staging on merge to `develop`
- Build and deploy to production on merge to `main`
- Send Slack notifications on failure

## Acceptance Criteria

- [ ] GitHub Actions workflow file created
- [ ] Tests run automatically on PR
- [ ] Staging deploys work
- [ ] Production deploys work
- [ ] Slack integration configured

# Todos

Todos let you track actionable items directly on issues and view them all in one place.

## Adding a Todo

1. Open any issue from the Board or List view
2. Scroll to the comments section at the bottom
3. Type your todo text in the comment box
4. Click **Add Todo** (next to "Add Comment")

The todo appears in the comments list with a checkbox. Regular comments and todos live side by side.

## Completing a Todo

Click the checkbox next to any todo to mark it as done. The text gets a strikethrough and the item dims. Uncheck to reopen it.

## Todos View

Click **Todos** in the top navigation bar to see all todos across all issues in one list.

Each todo shows:
- A checkbox to toggle completion
- The todo text
- A clickable link to the parent issue (e.g., "PROJ-rp-5: Fix login bug")

### Filtering

Use the segmented filter buttons at the top right:
- **Open** (default) — shows only incomplete todos
- **All** — shows everything
- **Closed** — shows only completed todos

Each button displays a count of matching items.

## Notes

- Todos are stored as comments, so they appear in the issue's comment history
- Todos persist in git like all other `.project` data
- The Todos view aggregates across all issues — useful for daily standups or personal task tracking
